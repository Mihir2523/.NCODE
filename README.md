# .NCODE

## An Fully Web Based Python Code Editor and Interpretur

* Pyodide for code Execution
* CodeMirror for Web Code Editor
* IDE Like environment
* Javascript for Calls and State Updation

## If You want to use this Project then Copy the code or clone the repository

1. <b>git clone https://github.com/Mihir2523/.NCODE.git </b>
2. Open edit.html On Browser or using anyb IDE with LiveSever
3. Run The code and Enjoy.

## Problems in this Projects

1. It does not allow user input for now
2. It has limited amount of Python Libraries like Numpy,Pandas,Matplotlib
3. ML and DL Libraries are not Included
